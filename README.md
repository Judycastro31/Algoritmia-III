# Algoritmia-III
Este proyecto corresponde a la Unidad III de Algoritmia, donde se implementan dos versiones del código de gráficos estilo "tortuga", mejorando progresivamente la versión inicial. El objetivo principal es ofrecer al usuario una interfaz amigable y atractiva para crear dibujos mediante comandos de movimiento y manipulación del lápiz.

## **El proyecto incluye:**

CodigoN1.java (Consola): Interacción mediante comandos de texto en la terminal.

GraficosTortugaGUI.java (GUI): Visualización de los dibujos en una ventana gráfica usando Swing.

Todos los códigos y el diagrama de flujo (diagrama_de_flujo.pdf) están dentro de la carpeta desarrollounidadIII.

## **Funcionalidades**

Seleccionar caracter de dibujo.

Girar a la izquierda (GI) o derecha (GD) en múltiplos de 45°.

Avanzar (AV) una cantidad de pasos en la dirección actual.

Activar lápiz (BL) para dibujar.

Desactivar lápiz temporalmente (SL).

Borrar posición actual (BO).

Mostrar dibujo en consola o GUI.

Salir del programa (0).

## **Tecnologías y herramientas**

Lenguaje: Java 100%

Bibliotecas: java.util, javax.swing, java.awt

IDE recomendado: IntelliJ IDEA, Eclipse, VS Code con extensión Java

Sistema operativo: Multiplataforma (Windows, macOS, Linux)

## **Instalación y ejecución**

### **1. Clonar el repositorio**

git clone (https://github.com/Judycastro31/Algoritmia-III.git)

cd GraficosTortuga/desarrollounidadIII

### **2. Ejecutar versión consola (CodigoN1.java)**
javac CodigoN1.java
java codigoN1
### **Ejecutar versión GUI (GraficosTortugaGUI.java)**

javac GraficosTortugaGUI.java
java GraficosTortugaGUI
Interactúa mediante el menú de la consola; los dibujos se mostrarán en una ventana gráfica.

## **Ejemplos de uso**       
Cambiar caracter de dibujo: * → opción 1

Girar 90° a la derecha: 90 → opción 3

Avanzar 5 pasos: 5 → opción 4

Activar lápiz: opción 5

Mostrar dibujo: opción 8 → aparece la ventana con la figura
